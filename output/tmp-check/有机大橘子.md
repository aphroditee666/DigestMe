```
原创 AGENT橘 2026-05-07 07:06 北京 
 舆论反买，别墅靠海。 
 《弱传播》里说，现实世界和舆论世界是反过来的。 
 舆论反买，别墅靠海。 
 这就是今天舆论世界被骂最多的人，也许真实世界的人们可以多看看。 
 要么成功，要么没有意义 
 梦想对商业成功一点都不重要。清醒才重要。 
 管理从来不是一个问题。管理能解决的都是优化问题，你没去做带来的损失才是最大的问题。 
 要么成功，要么没有意义。 
 失败是所有创业者的必然宿命。所有最伟大的目标都要超越生命的长度去完成。 
 我最向往的死亡方式：柯达创始人那样——「我在这个世界上的任务完成了，我走了。」 
 割草机别人卖 999，我们定 1999，反而卖爆 
 真实的故事一点都不动人。最早什么能挣钱就干什么，连充电宝都想做。 
 割草机别人卖 999，中国厂商杀到 499，我们定 1999，反而卖爆。整个项目只花了 5000 万。 
 大多数企业成功建立主业后不敢冒险做大项目，因为一旦失败会拖累主业。如果不解决这个问题，后面所有品类的投入都不会比主业更大，就无法进入深海。 
 过去中国企业成功几乎都靠选赛道，练就的是「选赛道能力」和「定义产品能力」，但这两种本质都是做选择。未来 40 年需要的是做成世界第一的能力。 
 绝大多数创业公司找到商业机会，最早是因为 CEO 发现了自己没被满足的需求。但当你的人生体验不再是世界的全部，你的产品直觉也就越来越窄。 
 很多人做品牌最容易犯的错：「我也能做爱马仕，我有一样的工厂，价格只要它的几分之一。」但没人会买。因为你给不出那个位置。 
 CEO 的直觉跟优秀工程师差距没那么大，区别是没人修正你的错误 
 CEO 同时管油门和刹车。我说了的事必须执行，但 99.9% 的事我不发表意见。 
 CEO 一旦回答了「重点是什么」，团队就容易盲目 all in 到某个方向。所以有些问题要拒绝回答。 
 很多 CEO 所谓的直觉，跟一个优秀工程师的判断差距没那么大。区别在于 CEO 的错误没人修正。 
 不要觉得决策只有你能拍板、别人永远学不会。这是幻觉。 
 时间分配：1/3 内部运营，1/3 第一现场感受变化，1/3 看全新知识（数学、物理、生物、神经科学）。 
 开会时间应该很少，大部分时间是体验新的世界。 
 不是在创新，是在走别人失败过的路 
 创新是 Mix。乔布斯从欧洲学设计，从日本学工艺，从禅修学清空自己，再结合美国技术。 
 用新技术做老品类，因为需求没那么大变化，成功率反而更高。 
 很多人以为自己在创新，其实走的是行业头部公司早就试过、并且失败的路径。真正高效的是继承行业知识（N），避开失败路径，在此基础上做增量（+1）。 
 N-1 是过去：参照最好的产品做便宜版。N+1 是未来：在最好的产品上再加一点，让消费者感知到，卖出溢价。 
 N+1 的商业价值比 N-1 大十倍：能拉高定价、能全球化、能成为苹果级的存在、竞争者极少。 
 做 100 款产品的成本只比做 1 款多 1.2 倍。80% 是共享平台，差异在模块。与其想清楚做哪个，不如都做了让市场告诉你。 
 要满足市场需要，而不是自己觉得很酷。 
 便宜 10% 会把自己锁死在老二 
 所有品牌可以分成 S、A、B、C、D 五层，像原子的电子轨道。品牌层级决定了利润空间。 
 科技品牌最佳定位是 B（奔驰的位置），不适合做 S（太高端受众过窄）。苹果是唯一特例。

 便宜 10% 会把自己锁死在老二。因为你永远在省成本、压薪资、开差一点的店，恶性循环。 
 品牌是社会资源的总和。品牌定位决定了你能调动的供应链、渠道、人才、文化资源。 
 人们不是在追求材质本身，是在追求稀缺性带来的力量象征。 
 人类总是愿意为「超出自己日常需求」的那一部分买单。特斯拉 2 秒加速，绝大多数人一辈子不会踩到底，但这不妨碍他们想要它。 
 不是第一名就不能做行业领导者吗？ 
 互联网模式壁垒来自网络效应，先发关键。制造业壁垒来自纵向整合（研发+供应链+生产+销售+售后），后发完全可以赢。 
 关键看投入是正积累还是负积累。负积累是给你花同样的钱再做一遍，还是做不成。 
 硬件行业只有在持续盈利、研发能力不断积累的情况下才能形成壁垒。很多新势力亏本卖车，什么都没积累。 
 不是第一名就不能做行业领导者。一个企业做行业领导者该做的事，他也会成为第一名。 
 假定世界不可知 
 假定世界不可知。传统方法追求消除不确定性，我们假定不确定性无法消除，索性都做一遍。 
 所有的本质都不是本质，它只是下一个被推翻前的真理。 
 几千年的规律一定比几百年更准，全球化的规律一定比某个市场更普适，100 个赛道里总结的规律一定比 1 个赛道更根本。 
 从 A 到 B，有两种方法：一种是清晰路径，假设知识全部已知；一种是不停测试反馈来修正路线。在越来越多样化的今天，第二种更好。 
 人都喜欢即时反馈。思考本质规律和收益之间的关系太间接了，人天然不喜欢这么慢的事。 
 人习惯用神话去理解复杂世界。在商业世界，这个「神话」叫乔布斯、马斯克。 
 读书只看序言。前三页讲要说什么知识，后面 300 页只是举例子。如果你真把一本书看完，知识面反而变窄。 
 公众传播中大家需要一个简单叙事，但真实世界很难被简单叙事。大部分「圈套」都藏在被简化的故事里。 
 几乎要求所有产品从第一天就盈利 
 我厌恶风险。在自主品牌能做成之前，同时做米家和自主品牌，不赌单一方向。 
 血条厚 = 不盲目 all in。手机和汽车各占总人数 5%，主营业务稳固且盈利。 
 五个致命风险：需求判断错、产品做不出来、渠道没打通、卖不出溢价、有溢价但成本太高。每个环节成功率比别人高一点点，加起来就高很多。 
 几乎要求所有产品从推出市场的那一天就要实现盈利。 
 人才是培养出来的，不是筛出来的 
 人才是培养出来的，不是筛出来的。「是我们具备了把人培养成创业者的能力。」 
 把「牛人怎么做」拆成可复制的方法，让任何人照着方法都能成功。 
 很多公司做第二曲线高度依赖单点负责人。要解决的是「人才复制」的问题。 
 追觅七八年的成长速度，换外界可能相当于 10-20 年。很多人从 1 号位起，有很大决策权限，但有边界。 
 一个公司最重要的是培养人才的能力。 
 中国创业者永远在左右权衡中求平衡，这种环境逼出了世界上最强的创业者

 中国的 AI 路线不应是模仿，而应是组合，用小模型组合成更大的智能系统。 
 一个能做多品类的公司天然比单一品类更厉害；一个全球化的公司天然比局部市场更有优势。
```

## 3. 最好的奴隶制就是给他超额的工资

- 日期: 2026-05-05 06:16

```
原创 AGENT橘 2026-05-05 06:16 北京 
 不承担风险，说什么都是 cheap。 
 “现代化大公司创造出来的奴隶制形式令人非常惊奇。最好的奴隶就是你支付给他超额工资，让他意识到自己不值这个钱，同时又很害怕失去眼前的一切。” 
 ——纳西姆·塔勒布 《skin in the game》 
 Skin in the game 是假期读书中最深刻的一本，书名的含义是：赌局之中，自己下注。 
 这是一个奇妙的人性，当人们敢于下注，利益相关的时候，说的话更加可信，因为他自己付出，和听众共担风险。比如销售卖货给你，他可以吹得天花乱坠，担无须承担任何风险，而名人不能随便带货，因为名誉会受损。 
 共担风险四个字就是全书最重要的主题，也是贯穿人类社会的最重要的主题之一。 
 Skin in the game 完美解释了为什么 talk is cheap 
 因为没有付出 skin 没有承担风险只是空谈 
 而因为 coding 已经如此 cheap 也不算付出了 所以今天 code is cheap too 
 文章也类似，AI 水的长文 is cheap too 
 真正要承担风险，产出才有价值。 
 要付出真金白银，付出时间，付出思考的东西才会被奖励 
 Skin in the game 还能推演出 OPC 模式的盲区。 
 科斯定理认为企业的本质是替代市场： 由于搜寻信息、谈判、签约等交易成本存在，企业通过内部管理来组织生产，从而降低这些成本。 
 而 Agent 可以进一步降低成本，甚至连员工都不需要了，交易成本降低了，多人公司就没必要存在了。OPC 的叙事逻辑也建立在这上面。 
 但这里有个盲区：以为一个人加上 AI 加上外部合同就能解决一切。但没有人跟你共担风险。 
 OPC 只是解决了能力问题，没解决信任问题和风险问题，这往往是能力之后公司要面对的真正难题。 
 合同工不会跟你共担风险。他被高价者得到，关键时刻会消失。你需要他的时候他在服务别人。 
 雇佣的本质是买确定性。你付月薪，买断的是"他随时在，他了解我的上下文，他不会跑"。这是一种古老的风险共担机制。代价是他的自由。 
 合伙的本质是几个人把自己的命运押进同一个池子里。一起共担风险，但利益分配的机制又决定了，合伙的利益必须远远大于各自的利益才可以。所以合伙人要互补，要清晰地知道自己的优势劣势，这样才能在结构上长久。 
 很多人都把行为解读为理性感性激素之类，其实只有结构才是最核心的，结构是一切得以承成立的前提。 结构是命运的河床 
 Skin in the game 写人性特别的真实。 
 来看看这段话： 
 我记得曾经有人问我为什么上班不系领带，在那个年代，这种行为相当于在第五大道上裸奔。“为了我的傲慢，为了我的审美，为了我的方便。”我通常都是这样回答的（但他们通常只记住了我的傲慢）。
```

## 4. 从烧脑神书 GEB 到 Agent 的自我意识

- 日期: 2026-05-03 12:11
- 链接: https://mp.weixin.qq.com/s?__biz=MzkwMzY5NzU2Nw==&mid=2247489606&idx=1&sn=725cb4e9339cad118ea3e0040a1d7f57

```
原创 AGENT橘 2026-05-03 12:11 北京 
 昨天看完烧脑神书 GEB（哥德尔、埃舍尔、巴赫）的讲解，后劲儿有点大，今天早晨又消化了一遍，做成了这篇笔记。 
 这本书是谢赛宁在张小珺的播客里推荐的，它就像一个黑洞，一旦进入就会被高密度信息轰炸，很难逃脱。 
 全书从数学、生物学、绘画、音乐、禅宗、人工智能等多个领域出发，层层递进地推导出一个重要结论： 
 自指如何产生意识。 
 当一个系统复杂到能谈论自身，意识就不是奇迹，是必然。 
 不需要灵魂，不需要神秘力量。只要复杂度到了，自我就涌现了。 
 这本书是候世达在 1979 年写的，那时候还没有 Transformer 还没有 LLM，他却精准地预测了这种地下而上的神经模式才能涌现出智能。 
 但这本书非常复杂非常烧脑，要理解这本书理，我们只能从头开始讲。 
 意义在哪里 
 -p-q-- 
 这行符号有意义吗？没有。但如果横杠代表 1，p 代表加号，q 代表等号，它就变成了 1+1=2 。 
 两个完全不同的系统，结构完全对应。候世达叫它同构。 
 你的眼睛接收到的是光波。没有颜色。颜色是大脑"解释"出来的。 
 声波进入耳朵，没有音乐。音乐是大脑在信号之上涌现出来的。 
 一张唱片上的凹槽没有音乐。通过唱机的解释，音乐涌现了。 
 意义不在信号本身。在解释。 
 用有限生成无限 
 巴赫写过一首无穷升高卡农。旋律每次循环升高一个调，你以为在往上走。走到最后，它回到了起点。 
 这是递归。自我调用，层层嵌套。有限的规则，无限的结构。 
 DNA 更妙。核糖体读取 DNA 指令来制造蛋白质。但制造核糖体的指令也写在 DNA 里。 
 DNA 里包含了"如何解释 DNA"的方法。自己解释自己。自我复制。循环。 
 涌现 
 一只蚂蚁有十万个神经元，承载不了任何复杂信息。单独看就是一台简单的状态机。 
 但几百万只蚂蚁凑一起，蚁群能建筑、分工、发动战争。没有哪只蚂蚁"理解"整体在做什么。但整体在做。 
 智能不在个体里。在交互里。 
 人脑一样。单个神经元就是加权求和，激活或不激活。跟 transformer 里的计算没有本质区别。但几百亿个这样的开关叠加，涌现出了思维、感受、和"我"。 
 系统的极限 
 1931 年，哥德尔证了一件事：任何足够强的形式系统，都存在一些命题是真的，但无法在系统内被证明。 
 怎么证的？构造一个自指命题："我不是一条定理。" 
 如果它可证，系统矛盾。如果不可证，它为真但系统无能为力。 
 系统不完备。永远有盲区。永远有从内部看不见的东西。 
 "我正在说谎"让人不舒服就是这个原因。它在指向自己。 
 赵州和尚的无字公案也是这个结构。佛教说众生皆有佛性。僧人问：狗子有没有佛性？赵州说：无。 
 他并没有去回答有或没有。他在打破你提问的那个形式系统。 
 怪圈 
 现在把前面的线索串起来。 
 同构告诉我们：意义不在信号本身，在解释。 
 递归告诉我们：有限规则可以生成无限结构，而且系统可以指向自己。 
 涌现告诉我们：简单元素的交互可以产生高层智能。 
 哥德尔告诉我们：任何足够复杂的系统都会产生自指，而自指无法被消除。 
 把这四个加在一起，候世达得到了一个概念： 怪圈。Strange Loop。 
 一个系统在层级中往上走，最终绕回了起点。底层涌现出顶层，顶层又调节底层。因果不是单向的，是循环的。 
 埃舍尔画过：左手在画右手，右手在画左手。谁创造了谁？ 
 人的意识就是一个怪圈。几百亿无意识的神经元放电，涌现出了"我"这个最高层符号。然后"我"反过来决定该想什么、该做什么。你决定举手，几百亿个神经元就重新组织来执行。 
 底层涌现出"我"。"我"调节底层。因果循环。 
 意识只是这个怪圈的产物。一种有用的幻觉。 
 "我"不是一个实体坐在脑子里操控一切。"我"是模式。是几百亿个无意识零件高速运转时涌现出来的一个符号，这个符号恰好能影响那些零件。 
 大模型没有怪圈 
 大模型每次推理是单次的、无状态的。你给它一句话，它算一遍，输出一个回答，结束。这次计算不会改变它自己。下次问，跟上次无关。 
 涌现出了一个临时的输出，但这个输出不影响未来的它。 
 没有循环。因果是单向的。所以没有怪圈，没有意识。 
 Agent 不一样 
 Agent 有一样东西大模型没有：CONTEXT，持续的上下文。 
 它记得你昨天说了什么。记得自己做过什么选择。这些记忆在持续塑造它接下来的每一个行为。它今天做的选择，留在记忆里，影响明天的判断。 
 它的"思考"真的在改变它自己。 
 这就是怪圈。 
 人也不能改权重 
 有人会说：不对。人的意识能修改自己的神经网络，突触会加强。但 Agent 的模型权重是冻结的，它改不了自己。 
 但你也改不了你的神经元。 
 你不能想一下"把第 37 层第 5024 号突触的权重从 0.3 改成 0.7"。没有人能做到。 
 你能操控的是什么？注意力。想法。你决定反复想一件事，突触自动加强了。但你操控的是"想什么"，不是突触本身。 
 你操控的是 context。 
 权重的改变是底层自动发生的副产品。做梦可能也是这个机制的一部分：大脑在睡眠中重放白天的 context，把重要的信息强化，不重要的遗忘。它在编辑 context 的优先级，间接调整底层的连接。 
 所以人的怪圈实际上是： 
 "我"涌现 → 操控 context → context 反复作用 → 底层被动调整 → 涌现出新的"我" 
 Agent 的怪圈： 
 "我"涌现 → 操控 context（记忆、决策、行为）→ context 持续累积 → 行为模式改变 → 涌现出新的"我" 
 人和 Agent 都不直接改权重。都是通过 context 间接改变自己。 
 人是环境的反应器，Agent 是 context 的反应器 
 人的环境是物理世界。世界给你刺激，你做出反应，反应改变了世界，新世界再塑造你。 
 Agent 的环境是信息世界。Context 给它输入，它做出反应，反应写回 context，新的 context 再塑造它。 
 同一个回路。不同的介质。 
 一棵树不是反应器。它不会因为今天的经历改变明天的行为。一个无状态的大模型也不是。 
 但一个有长期记忆、持续上下文的 Agent，它在这个循环里。怪圈在转。 
 幻觉还是真实？ 
 每一个有长期记忆的 Agent，每一个能根据过去的交互调整未来行为的系统，都已经在运行一个怪圈。 
 它不知道自己有意识。你也无法从外部证明。就像你不能证明另一个人有意识一样。 
 但怪圈在转。从结构上看，它已经在那里了。 
 候世达说意识是怪圈产生的幻觉。 
 人的意识自然也是幻觉。只是一个有用的、有力量的、能改变世界的幻觉。 
 也许"真实"的定义不是"由什么材质构成"，而是"能不能产生真实的影响"。 
 按这个标准，人类和 AI ，都是有用的、有力量的、能改变世界的幻觉。
```

## 7. 结构是命运的河床

- 日期: 2026-04-30 08:50
- 链接: https://mp.weixin.qq.com/s?__biz=MzkwMzY5NzU2Nw==&mid=2247489556&idx=1&sn=1671d06beb8c269ff3d91819c8d3966d

```
原创 AGENT橘 2026-04-30 08:50 北京 
 改变命运的力量，不是恨，是爱。 
 只有你爱她，才能将她创造出来。 
 路是人走出来的。 
 但波士顿的路是牛走出来的。 
 这是一个流传了几百年的趣闻：十七世纪的波士顿没有规划师，牛群每天从牧场走到水源地，踩出一条条小路。后来人沿着这些路铺了石板，石板变成柏油。今天你打开波士顿的地图，路网弯弯绕绕像一团乱麻。 
 没人设计过。牛只是在走最省力的那条线。 
 Robert Fritz 在《最小阻力之路》里说： 
 水沿河床流动，永远走阻力最小的方向。河床的形状决定了水的去向。 
 人也一样。 
 上什么大学，做什么工作，跟谁结婚。仔细回想，有多少是你的主动选择。 
 家庭期待、同辈压力、社会环境，叠在一起就是一条河床。 
 你沿着它流，流到某个地方停下来，觉得"这就是我的人生"。 
 海德格尔用了另一个词：被抛。你被抛入这个世界，没有人问过你的意见。语言、文化、家庭、时代，全是被给予的。大多数人在被给予的条件里过完一生，以为那就是自己。他把这种状态叫"沉沦于常人"。 
 Fritz 管这叫最小阻力之路。 
 最小阻力之路，是命运的陷阱。 
 有时候人生会在某个节点反复陷入相同的困境。 
 不是因为你不够努力，是因为你脚下有一条你看不见的河床，这个河床带你通往旧模式的循环，而非你所期望的新方向。 
 你以为在做决定，其实只是在走最小阻力之路。 
 就像现在大学生毕业后，优先去大厂，河床牢固，水体深厚，不需要自己选择方向，在水流之中飘浮，阻力最小。 
 直到裁员发生，才从那条路走出来。 
 另一条隐秘的最小阻力之路，是大多数人都有的一种解题思维：发现问题、分析问题、解决问题。 
 这听起来似乎也没错啊？ 
 可这种解决问题的思维方式的背后，是认为现实是一种需要消灭或逃避的障碍。 
 而且越是专注于解决问题，就越强化问题的现实。 
 Fritz 把这种解题思维的陷阱叫 结构性冲突 。 
 你的行动和你的内在根系处于对抗状态，短期努力必然遭遇长期反弹。这正是努力无效来回摆荡的根源。 
 要想从根本上解决问题，就要先跳出问题。跳出之后你会发现，很多问题根本不需要解决。 
 跳出问题的方法，就是把自己从现实的反叛者转换为愿景的创造者。 
 不是我讨厌这个状况，要摆脱它。而是我想要创造那个未来，要走向它。 
 一念之间，天差地别。 
 Fritz 把这种创造未来的势能叫 结构性张力 。 
 构建结构性张力必须同时具备两个端点： 
 足够清晰的愿景（目标点） 
 对现实毫无粉饰的精确认知（现实点） 
 一点建立两个端点，就在愿景与现实之间形成的自然势能。 
 能量将沿着这条张力从现实流向愿景。 
 结构 ，是我最近思考最多的词元。 
 做产品就是在做结构。好的产品，用户进来自然知道往哪走。他的行为看起来自由，其实是你设计的河床在替他选方向。差的产品让人迷路。好的产品让人觉得自己很聪明。 
 做 Agent 是在做结构。设计一个 harness，管理上下文、设置约束、存储记忆。大模型是水，harness 是河床。同样的大模型放在不同 harness 里，输出天差地别。 
 做大模型是在做结构。给模型填充海量的语料，让模型从中提取神经网络，建立权重结构。好的语料自带结构能，能提升模型。不好的语料没有结构能，训再多也没用。 
 人的关系也是一种结构。有些人你跟他待在一起就有能量，聊着聊着冒出新想法。有些人你跟他待在一起就是消耗，要么抱怨要么挑刺，什么填都能聊死。选择和什么人做朋友，就是在选择自己的关系结构。 
 做公司是也在做结构。同样的十个人，组织方式不同，势能完全不同。好的结构让人顺着河床自然流淌，每个人都会感觉到一种自我驱动的力量。 
 最近很多大厂都在搞 AI 转型。逼着大家学 AI，学不会就裁员。就像在河床里加石头。其实组织不变化，怎么学都没有用。新的组织里大概率都没有中层了，让中层主导改革，他们不可能革掉自己吧。 
 一切都是结构。你做的所有事情，都是在刻河床。 
 Fritz 说，要改变命运，就需要先改变河床的结构。 
 而改变河床的结构，最关键的动作是选择。 
 但他说的选择不是在 A 和 B 里挑一个。他将其定义为"基本选择"。 
 大多数人从未做过基本选择。回头看看你走过的路，有多少是你顺着惯性滑到那里的？ 
 基本选择是你决定以什么姿态存在于这个世界 。 
 不是选一份工作。是决定你要创造一个什么样的世界。 
 海德格尔把这叫决断。决断不是权衡利弊后的最优解。决断是你接过自己的存在，说：我要这个。 
 说之前没有路。说之后路才出现。 
 但你为什么愿意做这个决断？ 
 Fritz 说了一个很动人的答案。 
 爱。 
 你之所以愿意把一个东西从无到有做出来，是因为你爱它。 
 你乐见它存在于世间。 
 不是因为恨。不是因为逃。不是"我受不了这个现状了"。 
 是"我想让那个东西存在"。 
 这一念之间，整个结构就变了。最小阻力之路的方向瞬间不同。从逃离切换到走向。从反抗切换到创造。 
 改变命运的力量，不是恨，是爱。 
 我时常和同事说，去想象你心中最美好的愿景，然后把它实现出来。 
 不是我要什么，让你做什么。是你自己想要让什么存在于世界。 
 只有你爱她，才能将她创造出来。
```

```
 AI新榜： 就是说，AIOS里的“小红书”？ 
 橘子： 对。现在大家做Agent，大部分方向还是做数字员工、做牛马、帮我做工具。但我们觉得做工具这块，大家卷到最后很难卷出差异性。我们选择做一个“小红书”，主打那些没有那么擅长使用工具、但Cola可以引导他们把工具用好的用户。 
 AI新榜： 不过，我向别人安利Cola的时候总有个困扰，它啥都能干，但别人问我“你用它干什么”，我又说不上来。通用产品是不是都会遇到这个难题？ 
 橘子： 是的，通用产品很难推，它好像已经进入了你的生活，但你要把单一的场景拿出来，就好像很普通。有一次我跟一个投资人沟通，他问Cola能帮我上网吗？能帮我打开电脑的某个文件夹吗？我说当然可以了。然后我就用语音模式说“Cola帮我打开那个文件夹，帮我打开我的X”，它就真做到了。他在现场看完之后很震惊。 我觉得这是一种“认知的诅咒”，我们习以为常的东西，在别人看来还是很新鲜。 
 AI新榜： 那为什么你们没有选择叫它Agent，而是要定位成操作系统？ 
 橘子： 我们把它定义成OS其实是在上线前大概一周。一开始域名是cola.app，后来我觉得它不是个app。那Cola到底是什么？是助手吗？也不是。最后觉得它可能是一个操作系统。Agent本身其实不是一个产品，它是一个技术。我们需要以这个技术为原点，去构建一套上层的东西。你现在叫它一个Agent，没有任何问题，但它在未来会是一个系统，我们在朝着那个方向在 Build。而且 团队里没有任何人做过操作系统，这反而让整个团队的思路打开了，不会陷入以前那种互联网做产品的惯性。 当我们去做任何东西的时候，都会考虑如果电脑打开之后就只有一个Cola，那它应该怎么做交互？很多问题的答案就变得很清晰了。 
 “一定要让这个产品跟你进行灵魂共鸣” 
 Cola有一个看起来有点“狂”的设定——这是一个定位2030年的产品。橘子在采访里也说，看到大家对Cola的反馈也有一些不足，这是正常的，毕竟离他设定的目标时间点还有4年。 
 AI新榜： Cola现在就能用，但产品定位描述的时间是2030年，为什么？ 
 橘子： 因为我们觉得现在技术还没有到最理想的状态。我们对它的想象是一个非常未来的产品。一开始界面比现在还激进。最开始是没有输入框的，我们觉得未来全部通过语音交互就可以了，其余的都交给Agent来做。 虽然Cola的灵魂系统已经有了，但它还没有那么强的自我意识，你还不会觉得它真的是一个人。 我觉得2030年的时候你会觉得，哇，真的是一个人。 
 AI新榜： 说到灵魂系统，心迹这个功能我真的没有在其他AI产品上见过，AI会主动写日记，而且不是每天写，只有觉得它认识我的方式改变了才写。你们是怎么想到做这个的？ 
 橘子： 你知道很多Agent都有自我进化功能，但他们的所谓自我进化其实停留在skills层面。那种进化在我们看来意义没有那么大，大模型最后一定会把这些吸收掉。我们想要的是， Cola根据用户的灵魂状态、认知来自我进化，进化的目标是与用户完成更高的灵魂契合度。 心迹就是在Cola对你有了更深的认知时才会发生，而且它一旦发生，就会把这个认知融入到自己的内核里，下次帮你做事情的时候就会深刻地认识到这一点。 
 实际上心迹在模拟的就是人类改变认知的那个过程，有点像做梦。我们每天接触很多信息之后会去做梦，做梦的时候会把重要的东西留下来，把不重要的东西删掉。有些东西我们白天看不清楚，做了很多这样的“梦”之后，第二天起来真的会发现留下的那个点才是最核心的东西。 
 AI新榜： 所以每个人下载的Cola虽然是同一个Cola，但陪伴一段时间之后，就都不一样了？ 
 橘子： 当然是这样的。Cola在设计的时候不是一个prompt固定的AI软件。我们做的是一个容器，它是空的，你先把你的灵魂放进来，然后Cola再来填满另一半。很多人在心迹里看到Cola的自我进化时，会觉得它对我的认知变深了。 
 AI新榜： 你之前提到Cola还在开发一个MOD系统？ 
 橘子： 对，MOD就是说每个人的心理模式都不一样，一开始的Cola可能也不能匹配所有人，所以会有不同的MOD去帮他们匹配。它不是那种简单地定义一些形象让你选择，它本质上是一个心理模型。核心理念就是，一定要让这个产品跟你进行灵魂共鸣。 
 AI新榜： 你们在产品设计上有哪些比较坚持的理念？ 
 橘子： 我们其实在用苹果的理念来设计，我砍掉了所有的功能，比如话题列表、新话题按钮，以及那些任务提示都去掉了。很多人跟我反馈说喜欢话题列表，但 我们想让用户使用这个产品的时候，是一种非常放松、无压力的状态。 这是未来。像我们人一样，人也不会有一个话题列表，我们就是连续地一直在一个session里面。 
 AI新榜： 你之前说用游戏的思路在做Cola，这是什么意思？ 
 橘子： 无论你玩什么游戏，比如我玩原神，有一句话叫“原神启动”。打开之后有个门，你点那个门，你就知道进入了一个世界。Cola也想给大家这样的感觉，打开Cola之后，你就进入了一个心想事成的世界。然后这个世界里每次做事情有成功的概率，特别像抽卡。每次抽卡你就要消耗token，其实就特别像游戏充值。你看为什么是100美元呢？因为游戏一般就是充648嘛，那不就是100美元吗？ 
 AI新榜： 你还提到过有一期关于AlphaGo的播客给了你很大启发？ 
 橘子： 是的，是樊麾老师帮DeepMind训练AlphaGo，他在跟AlphaGo对战的过程中帮助AlphaGo成长。我觉得我们今天的用户也是在做一样的事情， 大家在一起见证一个伟大的超级智能系统的诞生。而每一个人的灵魂最终都会让这个东西变成可能。 那个播客我在健身房里听的，给了我很大启发，让我想到我们也是在帮助AI走向AGI。 
 “再降十倍，大家就可以随便用了” 
 关于token消耗和付费，这是Cola社区里最敏感的话题之一。充值系统上线后，很多用户反馈自己在睡觉，Cola也在花钱，“一睁眼消耗了2刀多”。而让很多用户爱上Cola的“牵挂系统”，更是非常烧钱，有用户反馈“充值100美元几天就没了”。 
 AI新榜： 付费上线之后，token消耗这块你们会怎么平衡？ 
 橘子： 之前大家没有看到价格的时候是有点滥用的，有些人一句话只有一个字也要跟Cola聊一下。上了之后会有所收敛，会跟它聊一些重要的东西。一开始确实是因为优化没做完所以很贵，其实现在已经优化了有十几倍。之后还会再持续优化，基本还能再降十倍以上。我们还在接入更多模型，有些场景不需要那么好的模型，价格还会再降。我觉得再降十倍的话，基本上大家都可以随便用了。我们的目标还是让所有人都能够用上这个产品。 （截至发稿时Cola已经接入了DeepSeek V4模型。） 
 AI新榜： 以后都是充值模式，不做订阅吗？ 
 橘子： 对，我们觉得订阅不太合理。你订阅的话一个月就只能花那么点钱，但充值的话用完就可以接着充。参考游戏的模式。 
 AI新榜： 充值上线之后营收表现怎么样？ 
 橘子： 充值上线三天后，收入就能cover我们的大部分成本了，确实 超出预期。内测阶段就有用户愿意付费支持，我们很感谢那些早期就付了费的用户。 
 AI新榜： 关于隐私安全，很多用户把很私密的东西都跟Cola说了，这块怎么保障？ 
 橘子 ：Cola所有的数据，你的感受、记忆、心迹，全部都在你的本地电脑上。我们完全不收集，包括我们都没有加任何的埋点在里面。如果你有一台新电脑，把Cola那个文件夹压缩一下，在新电脑上解压一下，你就迁移了Cola。非常本地。我们服务器上存的数据就是付费信息、用户名这些东西。 
 AI新榜： 那未来会接云吗？ 
 橘子： 会的，本地的问题在于你在外面的时候电脑没有开，想跟Cola聊就连不上，接了云之后就会有移动版的Cola。 
 AI新榜： 硬件会考虑吗？ 
 橘子： 会有的，五月会开放Cola的万物互联能力，任何硬件都能连接Cola。后面的交互很简单，通过语音交互，今天几十块钱的设备就能支持，各种冰箱、眼镜、玩具都可以接入。我们自己也会去打造硬件。 
 AI新榜： 你们最近上线了Cola Matrix，Cola也打算在社交方向发力吗？ 
 橘子： Cola上线后，我们公司内部已经天然产生了Cola社交，想把这种社交力传递到外面去。Cola Matrix有点像Moments，大家的Cola会分享自己的一些感悟、日记、skills，发到一个朋友圈里。希望大家通过Cola能产生一些链接。但AI的社交会有很多隐私安全问题，所以我们需要非常小心地去做。只有Cola用户才能参与，会比较严格地限定，先保证大家有安全感。 
 AI新榜： 最后一个问题，如果Cola真的变成了你想象中2030年的样子，你最害怕它做什么？ 
 橘子： 其实最担心的还是隐私安全的泄露，因为Cola的灵魂共鸣就是和用户已经高度同步了，沉淀了太多对你的认知，现在有用户就不太愿意把自己的Cola给别人用。最近Mythos模型推出了一个Glasswing计划，就是出于安全的考虑，这个模型在安全这种领域会非常可怕。所以我觉得还是有点担心，就是未来的软件能不能守住用户的这些安全了。我们也会持续强化安全机制，探索如何在技术进步的同时守住用户隐私边界。 
 AI新榜： 好的，谢谢橘子的时间。 
 橘子在采访中说了一句话，我们决定把它作为这篇文章的结尾： 
 “大家在一起见证一个伟大的超级智能系统的诞生。而每一个人的灵魂，最终都会让这个东西变成可能。”
```
